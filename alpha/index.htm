ClipBoard-2000 Alpha Test

This is an Alpha test of ClipBoard as it develops. We don't release software which we know to work badly, but we do some basic testing and out it goes.

Generally we are surprised by crashes. Often it is a hardware or device driver that we have never encountered. So if you see a problem, carefully reproduce it, record the configuration, and send us a note.

The team members now working on ClipBoard-2000 include:

  • Drew Daugherty - PC Version Developer (drewid@umich.edu)
  • Phil Trieb - Mac Version Developer (treib@umich.edu)

Before you Download:

The ZIP or SIT files have a password which you will receive when you join my mailing list by sending a message to cb2k-subscribe@eGroups.com. The initial message will contain the password. I track this so I can tell you of new versions as they are released. Some allow download and others do not. Generally you should take the latest version. I remove the download when a version has a nasty bug which has been fixed in a later release.
